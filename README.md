# Svelte + TS + Tailwind

This template help get you started developing with Svelte + TS + TailwindCSS and Routing.

### This Repo Provided

- Svelte
- TypeScript
- Svelte-Routing(SPA)
- TailwindCSS
- Vercel Deploy
- pnpm

### Usage

- To install package

```bash
pnpm install
```

- To start development

```bash
pnpm dev
```

- To build production

```bash
pnpm build
```
